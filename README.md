# いいわけ

本家にバグがたくさんあり、文字列を変更できない場所や、簡体字入力手段を持ってないので、
追従できることが限られます。<br>
また中華フォントも、ソースがありませんので、変更できません。

元々のソフトがバイナリのみの配布ですので、使用にはご注意ください。<br>
日本語部分を含めて自己責任でお願いします。
